# Social and Utility APIs

## `/api/campaigns/[id]/supporters`

Returns public-safe supporter data for a campaign.

## `/api/badges/[userId]`

Returns user badge data.

## `/api/bookmarks/check`

Checks whether the current user bookmarked a post.

## `/api/follows/check`

Checks whether the current user follows a target.

## `/api/posts/[postId]/comments`

Returns comments for a post.

## `/dashboard/giving/financial-year`

Returns financial-year giving data for the signed-in supporter.

## `/ngo/dashboard/tax/10bd`

Returns NGO tax export data for Form 10BD support.
